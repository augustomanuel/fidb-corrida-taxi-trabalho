# 🚖 Cálculo de Corrida de Táxi em PHP

Este projeto consiste num sistema simples desenvolvido em **PHP** para calcular o valor de corridas de táxi, utilizando lógica de funções e procedimentos para organizar a execução.

## 📝 Sobre o Projeto

O programa processa o custo de uma viagem com base em parâmetros pré-definidos, estruturado da seguinte forma:

*   **Função `calcularCorrida`**: Responsável por realizar os cálculos matemáticos do valor da tarifa (ex: valor por KM + taxa fixa).
*   **Procedimento `exibirCorrida`**: Responsável por formatar e mostrar o resultado final ao utilizador no ecrã.

## 🛠️ Tecnologias Utilizadas

*   **Linguagem:** PHP 8.x (ou superior)
*   **Ambiente:** CLI ou Servidor Web (Apache/Nginx)
